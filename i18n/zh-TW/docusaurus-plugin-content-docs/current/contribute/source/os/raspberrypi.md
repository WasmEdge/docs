# 在 Raspberry Pi 上建置

Raspberry Pi 自 3 Model B 起即使用 64 位元處理器。因此 WasmEdge 也可以在 Raspberry Pi 上執行。對於 ARM,您可以選擇任何 64 位元 Linux 發行版,例如 Raspbian、Ubuntu 或 Manjaro。本文件已在 Manjaro for ARM 發行版上進行測試,硬體為 Raspberry Pi 3 Model B。

安裝步驟與[安裝文件](../../../start/install.md#install) 並無不同,執行方式也相同。以下是一段安裝 WasmEdge 並執行簡單 WebAssembly 模組將兩個數字相加的影片。
