# 加入 WasmEdge 社群

歡迎每一位朋友加入 WasmEdge 社群。

## 聯絡方式

如果您有任何問題,歡迎在相關專案上開立 GitHub 議題,或加入下列管道:

- 郵件論壇:寄送電子郵件至 [WasmEdge@googlegroups.com](https://groups.google.com/g/wasmedge/)
- Discord:加入 [WasmEdge Discord 伺服器](https://discord.gg/h4KDyB8XTt)!
- Slack:加入 [CNCF Slack](https://slack.cncf.io/) 上的 #WasmEdge 頻道
- Twitter:在 [Twitter](https://twitter.com/realwasmedge) 上追蹤 @realwasmedge

## 社群會議

我們每月舉辦一次社群會議,展示新功能、示範新的使用案例,並安排問答時間。歡迎所有人參加!

時間:每月第一個星期二香港時間晚上 11 點 / 太平洋標準時間早上 7 點。
