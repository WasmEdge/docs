# WasmEdge 入門

WasmEdge 是一個輕量、高效能且可擴充的 WebAssembly 執行環境,專為雲原生、邊緣與去中心化應用所打造。它支援無伺服器應用、嵌入式函式、微服務、使用者自訂函式 (udf)、智慧合約以及 IoT 裝置。WasmEdge 目前是 CNCF (Cloud Native Computing Foundation) 的 Sandbox 專案。

WasmEdge Runtime 為其所包含的 WebAssembly 位元組碼程式提供明確定義的執行沙箱。此執行環境提供作業系統資源(例如:檔案系統、socket、環境變數、行程)與記憶體空間的隔離與保護。WasmEdge 最重要的使用情境是在軟體產品(例如:SaaS、資料庫、邊緣節點,甚至是區塊鏈節點)中以外掛形式安全地執行使用者自訂或社群貢獻的程式碼。它讓第三方開發者、廠商、供應商與社群成員能夠擴充並客製化軟體產品。
